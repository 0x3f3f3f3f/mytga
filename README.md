# My 2024 The "fuck you official TGA" Game Awards

一个用来制作个人版游戏年度评选的网页工具。

## 💻 技术栈

- [Astro](https://astro.build)
- html2canvas (截图功能)

## 🚀 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📝 使用说明

1. 在左侧区域粘贴游戏截图
2. 在右侧填写奖项名称和感想
3. 点击右上角的"下载截图"按钮保存
4. 分享你的个人年度游戏大奖！
